# Data-Monster

## Description 

This project is a Python program that is able to take in an Excel file (.xlsx) and then do either of the following:
- Clean the data
- Analyze the data
- Visualize the data
- Export the data to a new Excel file (.xlsx)
- Export the data to a new CSV file (.csv)
